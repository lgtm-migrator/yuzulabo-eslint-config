# yuzulabo-eslint-config

> A shared [eslint](https://eslint.org/) config for my projects.

## Installation

- Install `@yuzulabo/eslint-config` via npmjs.com

```bash
yarn add @yuzulabo/eslint-config --dev
```

- Import config in your `package.json`

```
{
  ...
  "eslintConfig": {
    "extends": "@yuzulabo"
  }
  ...
}
```
